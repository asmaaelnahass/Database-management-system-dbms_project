# Create Database management system (DBMS)

Create Database management system(DBMS) using Bash-Script A simple Database Management System using bash shell scripting that handles all CRUD operations.

# Features

 * Create Database
 * Connect Database
 * Drop Database
 * List Databases
 * Create Table
 * Insert into tables
 * Select from Table
 * Delete from Table
 * Update Table
 * Drop Table
 * List Tables
